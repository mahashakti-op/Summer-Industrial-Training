# 💰 Medical Insurance Cost Prediction using Decision Tree Regressor

## 📌 Project Overview

This project predicts the estimated medical insurance charges based on a person's demographic and health-related information using a Decision Tree Regressor.

Here's the link : https://summer-industrial-training-h5nccjzgptzwnp7ihm5pbd.streamlit.app/
---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit
- Joblib

---

## 📂 Dataset

**Medical Insurance Cost Prediction Dataset**

### Features

- Age
- Sex
- BMI
- Number of Children
- Smoker
- Region

### Target

- Insurance Charges

## 🌐 Streamlit Application

The application allows users to enter:

- Age
- Sex
- BMI
- Number of Children
- Smoker Status
- Region

and predicts the estimated medical insurance charges.

## 🚀 How to Run

Clone the repository

```bash
git clone <repository-link>
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
