# 📅 Day 05 – Data Visualization with Matplotlib & Seaborn

**Date:** **09 July 2026**

Day 5 focused on learning the fundamentals of **Data Visualization** using Python. I explored **Matplotlib** and **Seaborn**, two of the most widely used libraries for creating statistical and analytical visualizations.

---

## 📚 Topics Covered

### 📊 Matplotlib

- Introduction to Matplotlib
- Creating basic plots
- Understanding different chart types
- Customizing graphs
- Adding titles, labels and legends

#### Visualizations Explored

📈 **Line Chart**
- Used to visualize trends over time.

📊 **Bar Chart**
- Used to compare values across different categories.

📉 **Histogram**
- Used to visualize the frequency distribution of numerical data.

🥧 **Pie Chart**
- Used to represent proportions or percentage distribution.

🔵 **Scatter Plot**
- Used to study the relationship between two variables.

📦 **Box Plot (Whisker Plot)**
- Used to summarize data distribution, median, quartiles and detect outliers.

---

### 🎨 Seaborn

- Introduction to Seaborn
- Advantages over Matplotlib
- Creating attractive statistical visualizations
- High-level plotting interface
- Better styling and built-in themes
