# ❤️ Heart Disease Prediction using Decision Tree Classifier

## 📌 Project Overview

This project predicts whether a patient is likely to have heart disease using a Decision Tree Classifier.

Here's the link :  https://summer-industrial-training-r37kd57e63eachm4zstvz9.streamlit.app/

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

## 📂 Dataset

Heart Disease Prediction Dataset (Kaggle)

## 📊 Machine Learning Workflow

- Data Loading
- Exploratory Data Analysis (EDA)
- Data Cleaning
- Target Encoding
- Train-Test Split
- Model Training
- Model Evaluation
- Streamlit Web Application

## 📈 Algorithm

Decision Tree Classifier

## 🚀 How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 📷 Output

The application predicts:

- ❤️ High Risk of Heart Disease
- ✅ Low Risk of Heart Disease
